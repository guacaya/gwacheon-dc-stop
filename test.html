🚀 과천 주암 데이터센터 반대 민원 안내

국민신문고 민원을 대량 접수하거나, 문자 전송 버튼으로 관계자들에게 글자 잘림 없이 항의할 수 있습니다.

⚠️ 꼭 읽어주세요!
이 페이지는 민원 내용을 자동으로 발송하지 않습니다.
부처 버튼을 누르면 민원 텍스트가 생성되고, 이를 복사해서
국민신문고에서 본인이 직접 로그인 후 접수해야 합니다.
📱 홈 화면에 추가해두면 다음부터 한 번에 접속돼요
아이폰 — 하단 공유 버튼(⬆️) → 홈 화면에 추가
안드로이드 — 우측 상단 메뉴(⋮) → 홈 화면에 추가 또는 앱 설치

👆 위 내용을 확인하고 아래에서 민원을 접수해 주세요!
📋 민원 접수 순서 (처음이신 분)
1
국민신문고 회원가입 및 로그인
본인인증(카카오·PASS 등)이 필요합니다.
2
아래에서 부처 탭 선택 → 제목/본문 복사
탭을 누르면 수신기관명이 자동 복사됩니다.
3
국민신문고 → 일반민원 → 수신기관 검색 후 붙여넣기
수신기관 검색창에 복사된 부처명을 붙여넣고, 제목·본문도 붙여넣기.
4
접수 완료! → 다음 부처 탭 선택 반복
임시저장 기능으로 빠르게 여러 개 접수 가능 (아래 꿀팁 참고).
⚡ 1분에 10개 접수하는 임시저장 꿀팁
  1. 민원 내용 붙여넣기 후, 페이지 맨 아래 [임시저장] → [홈화면 바로가기]
  2. 홈에서 [일반민원] 재클릭 → 개인정보 입력
  3. 우측 [불러오기] 클릭 → 직전 임시저장 내용 자동 로드
  4. 제목·본문만 새로 교체 후 접수 → 2~4번 반복!
TIP 1 민원 공개 설정 (강추): 작성 페이지 하단에서 '공개'로 설정하면 목소리가 더 크게 퍼집니다.
TIP 2 주소 미리 등록: 국민신문고 내 정보 메뉴에서 거주지 주소를 미리 등록해두면 매번 입력 불필요.
동 💡 입력하지 않아도 문자 발송은 가능합니다. 단지와 동을 선택하시면 문자 첫 줄 소속 정보가 자동으로 자연스럽게 변환됩니다.
